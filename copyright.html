Copyright

Copyright dient zum schutz von Geistigem eigentum. Anderst als das Urheberrecht, geht es beim Copyright nicht primär um den schutz des Urhebers, sondern darum, dass geistifes Eigentum nicht ohne Erwähnung oder zustimmung verfielfältigt wird. Darum kann zum schützen von geistigem Eigentum das Copyright Zeichen © verwendet werden.
Auf dieser Website sind zum Beispiel die eingefügten Bilder durch Copyright geschützt. Darum sind alle Bilder mit einer Quelle angegeben, um den Urheber des Bildes zu erwähnen.

Copyleft

Neben dem Copyright gibt es auch noch das Copyleft. Copyleft ist eine Lizenz, welche oft im Software bereich für open sourve code verwendet wird. Software welche mit Copyleft Lizenzen veröffentlicht werden, können verwendet und verändert werden. Voraussetzung für die verwendung ist aber, dass das daraus entstandene Produkt auch wieder mit einer Copyleft lizenz veröffentlicht wird, so dass andere Personen die erweiterte Software auch verwenden können.

Zu den Copyleft lizenzen zählen unter anderem:

  • Mozilla Public License (MPL)
  • Lesser General Public License (LGPL)
  • GNU general Public License (GPL)

Lizenzen

Das Wort Lizenz kommt vom lateinischen "licet" was so viel wie "ist es erlaubt" bedeutet. Das beschreibt auch ziemlich treffend was Lizenzen sind, und wie diese Verwendet werden. Mit Lizenzen wird geregelt, wie Content im Internet verwendet und dupliziert werden kann. Zu den Lizenzen zählen zum beispiel das oben beschriebene Copyleft. Allerdings gibt es noch andere Formen der Lizenzen. Zum beispiel können Lizenzen auch nur an einzelne Personen verkauft werden. Dabei kauft der Nutzer nicht die Software selbst, sondern nur die Erlaubniss die Software zu nutzen. Ein Beispiel dafür ist zum Beispiel photoshop. Beim Kauf einer Photoshop Lizenz kaufe ich nicht das Programm selbst, sondern nur die Erlaubniss die Software für einen bestimmten Zeitraum zu nutzen.

Public Domain

Als Public Domain werden Inhalte bezeichnet welche nicht durch Lizenzen geschützt sind. Sogenanntes Allgemeingut kann nicht Lizensiert werden und dadurch auch nicht kostenpflichtig zum Verkauf angeboten werden. Kurz gesagt: Jeder darf es benutzen und die Inhalte gehören der Allgemeinheit.

Fair Use

Als Fair use wird eine regelung bezeichnet, welche zulässt, dass Inhalte welche Urheberrechtlich geschützt sind dennoch verwendet werden können wenn es sich zum Beispiel um kritik oder Medien Artikel handelt. Was allerdings als Fair Use gilt, muss im streitfall ein gericht entscheiden. Das Fair Use Prinzip wird vor allem in Amerika verwendet.
