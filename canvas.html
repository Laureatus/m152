Canvas

HTML Canvas kann verwendet werden um mit Javascript Grafiken auf einer Website zu Zeichen.

Beispiele:

Rechteck


id="myCanvas" width="200" height="100" style="border:1px solid #000000;

Dies ist ein simples Beispiel für eine Canvas. Innerhalb dieser Kann nun mit JavaScript Code gezeichnet werden.

Zeichnen von linien


var c = document.getElementById("myCanvas");
var ctx = c.getContext("2d");
ctx.moveTo(0, 0);
ctx.lineTo(200, 100);
ctx.stroke();

Zeichnen von Kreisen


var c = document.getElementById("myCanvas");
var ctx = c.getContext("2d");
ctx.beginPath();
ctx.arc(95, 50, 40, 0, 2 * Math.PI);
ctx.stroke();

Zeichnen von Text


var c = document.getElementById("myCanvas");
var ctx = c.getContext("2d");
ctx.font = "30px Arial";
ctx.fillText("Hello World", 10, 50);
