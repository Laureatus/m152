Reflexion

Wireframes und Mockups

Das Erstellen von Wireframes und Mockups ist mir leicht gefallen, da ich bereits erfahrung mit Figma durch vorherige Module hatte. Darum habe ich in diesem Punkt nicht sehr viel dazu gelernt.

Styleguide

Auch bei den Styleguides habe ich nicht viel recherche gebraucht, um zu wissen, was ich zu tun hatte. Im Modul User Expierience und User Interface hatte ich den grössten Teil schon gelernt.

Urheberrecht

Im bereich Urheberrecht habe ich einiges dazu gelernt. Ich hatte zwar im Allgemeinbildenden Unterricht schon ein Paar Dinge zu diesem Thema gelernt, dennoch konnte ich mein Wissen eerweitern.

Copyright

Im Bereich Copyright Hatte ich noch keine erfahrung. Darum habe ich hier sehr viel dazu gelernt, welche Arten von Copyright es gibt, und wie ich Bilder und andere Ressourcen verlinken muss, um sie auf meiner website einzubinden.

Datenschutz

Im Bereich Datenschutz konnte ich viel dazu lernen. Dass es so viele bestimmungen und Vorgaben gibt um Daten zu schützen, wusste ich nicht. Das Thema war sehr intressant.

Bilder

Über die Bilder wusste ich das meiste schon. Ich brauchte allerdings sehr lange für diese Aufgabe, da das Beschreiben der verschiedenen Bild, Video und Audi Formate sehr lange gedauert hat.

CSS Animationen

Ich kannte bisher nur die CSS Animationen, welche mit :before und :after gemacht werden können. Dass es noch so viele weitere Möglichkeiten gibt, wusste ich nicht. Ausserdem konnte ich mich mal mit Java Script beschäftigen und eine Rekursive Funktion implementieren, welches ein berühmtes Geometrisches Muster zeichnet.

Gesamteindruck

Alles in allem, konnte ich sehr viel im Bereich Webentwicklung im Frontend dazu lernen, da ich normalerweise nur Backend Entwicklung betreibe. Einige Aufgaben waren sehr zeitaufwändig, habe aber dennoch Spass gemacht.
